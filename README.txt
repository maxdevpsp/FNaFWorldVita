Управление:
- В меню:
  - КРЕСТ - перейти к бою
  - ТРЕУГОЛЬНИК - показать титры
	- КРУЖОК - выйти из титров
	
- В игре:
  - Нажатия на кнопки происходят через экран.
  - Выход через меню Виты.
  
Controls:
- In menu:
  - CROSS - start the fight
  - TRIANGLE - show credits
	- CIRCLE - exit from credits
	
- In game:
  - Tap to interact with buttons
  - Exit through Vita's menu
